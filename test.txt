JavaScript (Explain + Show Code)
1. Closure
=>  Function remembers its outer scope even after execution
function outer() {
  let count = 0;

  return function inner() {
    count++;
    console.log(count);
  };
}
const fn = outer();
fn(); // 1
fn(); // 2

function outer2() {
  let count = 0; 
 function inner() { 
    console.log(count);
  };
  inner()
}
outer2()

Closure means function + its lexical environment

4. Arrow vs Normal Function
